\index{x86!\Instructions!RET}
\item[RET]: \IFRU{возврат из процедуры}{return from subroutine}: \TT{POP tmp; JMP tmp}.

\IFRU{В реальности}{In fact}, RET 
\IFRU{это макрос ассемблера, в среде Windows и *NIX транслирующийся в}
{is a assembly language macro, in Windows and *NIX environment is translating into}
RETN (``return near'') 
\IFRU{ибо, во времена MS-DOS, где память адресовалась немного иначе}
{or, in MS-DOS times, where memory was addressed differently}
(\ref{8086_memory_model}), \IFRU{в}{into} RETF (``return far'').

\TT{RET} \IFRU{может иметь операнд}{may have operand}.
\IFRU{Тогда его работа будет такой}{Its algorithm then will be}: \TT{POP tmp; ADD ESP op1; JMP tmp}.
\TT{RET} \IFRU{с операндом обычно завершает ф-ции с соглашением о вызовах \IT{stdcall}, см. также}
{with operand usually end functions with \IT{stdcall} calling convention, see also}: \ref{stdcall}.
