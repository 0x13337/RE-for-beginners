\index{MIPS!\Instructions!MFC1}
\RU{Новая для нас здесь инструкция это}
\EN{New instruction for us here is} MFC1 (``Move From Coprocessor 1'')\RU{ (копировать из первого сопроцессора)}.
\RU{FPU это сопроцессор под номером 1, вот откуда ``1'' в имени инструкции.}
\EN{FPU is coprocessor number 1, hence ``1'' in instruction name.}
\RU{Эта инструкция переносит значения из регистров сопроцессора в регистры основного CPU (\ac{GPR}).}
\EN{This instruction transfers values from coprocessor's registers to main CPU registers (\ac{GPR}).}
\RU{Так что результат исполнения pow() в итоге копируется в регистры \$A3 и \$A2,
и из этой пары регистров \printf возьмет его как 64-битное значение типа \Tdouble.}
\EN{So the result from pow() execution is finally moved into registers \$A3 and \$A2, 
\printf will take 64-bit double value from this registers pair.}
