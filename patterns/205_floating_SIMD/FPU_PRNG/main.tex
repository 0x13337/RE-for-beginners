\RU{У всех инструкций суффикс -SS, это означает}\EN{All instructions has -SS suffix, this mean} ``Scalar Single''.
``Scalar'' \RU{означает что только одно значение хранится в регистре}\EN{mean that only one value is stored in register}.
``Single'' \RU{означает что это тип \Tfloat}\EN{mean \Tfloat data type}.
